# 🛒 FreshMart – Online Grocery Store Management System

FreshMart is a web-based grocery store management system developed using **Java Spring Boot MVC architecture**.  
The system allows users to browse grocery products, manage cart items, place orders, and enables admin to manage products efficiently.

---

## 📌 Project Objective

The objective of FreshMart is to develop an efficient and user-friendly online grocery management system that simplifies product browsing, ordering, and inventory handling through a web-based platform.

## 🚀 How to Run the Project

Follow these steps:

### Step 1

Clone repository

git clone https://github.com/Harshadakorhale/FreshMart---Online-Grocery-Store-Management-System.git

### Step 2

Open project in **Spring Tool Suite (STS)**

### Step 3

Create database in MySQL

freshmart_db

### Step 4

Update database credentials inside

application.properties

Example:
spring.datasource.username=root
spring.datasource.password=yourpassword

### Step 5

Run project as
Spring Boot App
